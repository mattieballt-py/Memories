# Splat App

This is a [Next.js](https://nextjs.org) project bootstrapped with `create-next-app`, featuring 3D Gaussian Splatting capabilities.

## Features

- **Next.js 14.2.35** with App Router (security patched version)
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **React Three Fiber** for 3D rendering
- **React Three Gaussian Splat** for Gaussian Splatting
- **Three.js** for 3D graphics
- **Modal Client** for backend triggers

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

```bash
npm run build
npm start
```

## Deploy on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

```bash
vercel --prod
```
